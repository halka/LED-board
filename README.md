# LED電光掲示板

ブラウザで動作する LED 電光掲示板です。

## 機能

- 背景色と文字色の変更
- 文字の横スクロール
- ライブプレビュー
- PNG / WebP 画像ダウンロード
- WebM / MP4 動画ダウンロード
- `index.html / style.css / script.js` の分割構成
- スマホ向けの大きめUI、縦並びレイアウト、sticky保存エリア

## 使い方

1. `index.html` をブラウザで開きます。
2. テキスト、色、サイズ、速度などを調整します。
3. プレビューを確認しながら PNG / WebP / WebM / MP4 で保存します。

## 補足

- MP4 変換は `ffmpeg.wasm` を利用します。
- 初回の MP4 出力時は CDN から関連ファイルを読み込むため少し時間がかかることがあります。
- WebM や WebP の対応状況はブラウザごとに異なります。
