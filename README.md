# LED電光掲示板

ブラウザで動作する LED 電光掲示板です。

## 機能

- 背景色・主文字色・アクセント色・備考色の変更
- 横スクロールモード
- 鉄道駅の発車標のような駅風発車標モード
- 駅風オレンジ / 駅風グリーン / 駅風警告レッドのテーマ切り替え
- 備考欄の点滅表示
- ライブプレビュー
- PNG / WebP 画像ダウンロード
- WebM / MP4 動画ダウンロード
- `index.html / style.css / script.js` の分割構成
- スマホ向けの大きめUI、縦並びレイアウト、sticky保存エリア

## 使い方

1. `index.html` をブラウザで開きます。
2. 表示モードを `駅風発車標` または `横スクロール` から選びます。
3. テキストや発車標の内容、色、サイズ、点滅間隔などを調整します。
4. プレビューを確認しながら PNG / WebP / WebM / MP4 で保存します。

## 駅風発車標モード

以下の項目を設定できます。

- 種別
- 行先
- 発車時刻
- 両数
- 備考
- 備考欄点滅
- 駅風テーマ

## 補足

- MP4 変換は `ffmpeg.wasm` を利用します。
- 初回の MP4 出力時は CDN から関連ファイルを読み込むため少し時間がかかることがあります。
- WebM や WebP の対応状況はブラウザごとに異なります。
